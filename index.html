Projects

Habits

habits.email is my current project. It’s a dead simple habit tracking app that works mostly through your email. It’s so early that it doesn’t even have a proper name or logo yet, but people are already finding it useful!

testdouble-chai

testdouble.js is a mock object library for JavaScript. I maintain testdouble-chai, which makes testdouble.js a little bit nicer to use alongside the Chai assertion library.

Graph of Life

Graph of Life is a browser bookmarklet that animates the GitHub contributions graph with Conway’s Game of Life. It’s a silly little toy, but I figured I’d drop it here for good measure :)

Trello2Markdown

Trello2Markdown is a teensy little util that will take a Trello board (either as exported JSON or as a public URL) and convert it into a Markdown document. Great for people who like to outline text with Trello before publishing! There are dozens of us!

Delicious Ships

Delicious Ships! is a game that my friend Josh and I made for Ludum Dare. It uses PuzzleScript, which is a super fun little game engine you should check out it you haven’t!

BasecaseMusic

My neglected SoundCloud account. I started 2016 with the goal of releasing a new track every day, but, uh...lol. My hope is to get back into it (at a much-reduced pace) by the end of the year.
