Delicious Ships!

Delicious Ships! is a short puzzle game I made with my friend Josh for the 33rd Ludum Dare game jam. We designed and developed this game in only one weekend, September 21-23, 2015. Neither of us had ever made a game before this point, so it was a fun learning experience! We used the super cool PuzzleScript game engine.
