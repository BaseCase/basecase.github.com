Trello2Markdown

I like using Trello boards to organize writing, but it’s hard to go from a Trello board to an actual document. So I made Trello2Markdown to make that conversion super easy! It’s pretty barebones right now, but I have lots of features I’m planning to add.

#BaseCaseMusic2016

I’ve been a musician my whole life, but always as a performer and never as a composer. This year I decided to learn how to make music on the computer, so I’m releasing one short track every week as I learn to use Ableton Live. They aren’t very good (yet) but I’m shooting for quantity over quality just to get lots of practice under my belt!

testdouble-chai

testdouble-chai is an itty-bitty JavaScript library that adds a couple of handy assertions to Chai, which is my favorite assertion library for JS testing. It’s meant for use with the very nice test double library, testdouble.js.

Delicious Ships!

Delicious Ships! is a short puzzle game I made with my friend Josh for the 33rd Ludum Dare game jam. We used the super cool PuzzleScript game engine.
